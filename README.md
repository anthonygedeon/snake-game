<h1 align="center">Snake</h1>

## Installation

Install via pip:
```
pip install -r requirements.txt
```

## Play:
```sh
$ cd path/to/snake/dir
$ python3 snake.py
```

## Controls

- UP: Up Arrow Key
- DOWN: Down Arrow Key
- LEFT: Left Arrow Key
- RIGHT: Right Arrow Key
